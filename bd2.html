Bases de Datos 2

En informática, una base de datos es una colección organizada de datos estructurados, almacenada electrónicamente en un sistema informático. Se utiliza para almacenar, recuperar y gestionar información de manera eficiente. Las bases de datos pueden contener diversos tipos de datos, como texto, números, imágenes, videos y archivos.
